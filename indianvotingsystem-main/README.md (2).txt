Indian Voting System Research Project

This project focuses on improving voter confidence in the Indian Electronic Voting Machine (EVM) system using digital verification and structured data systems.
Features

- Voter Registration Portal
- Polling Office System
- Head Office Monitoring
- Database Validation
- SMS Authentication using Twilio
